### 🔎 문제 접근
N개의 수가 주어졌을 때, i번째 수부터 j번째 수까지의 합을 구하는 문제다.
매 쿼리마다 직접 합을 구하면 시간초과가 발생 하기에
누적 합 배열을 미리 구성하여 O(1)로 구간 합을 구하는 방식으로 풀었다.

<br />

### 📝 사용한 자료구조
- BufferedReader / BufferedWriter
- int[] arr
  - 원본 저장
- int[] s
  - 누적 합 배열 저장.
  - s[i] = s[i - 1] + arr[i] 공식 사용
  - 구간 합 s[end] - s[first - 1] 사용
    
<br />

###  🦾 개선 포인트
- 변수명 개선
- arr 배열은 누적 합 배열 구성 후 사용하지 않으므로 제거 가능
    
<br />

###  💡 배운 점
구간 합을 매번 직접 계산하면 시간 초과가 발생할 수 있으며, 누적 합 배열을 미리 구성해두면 O(1)로 빠르게 구간 합을 구할 수 있다.
누적 합 공식 S[j] - S[i-1] 을 활용하면 임의의 구간 합을 효율적으로 처리할 수 있다.
