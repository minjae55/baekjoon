### 🔎 문제 접근
각 숫자의 차이가 일정한 수 갯수 구하기

<br />

### 📝 사용한 자료구조
- int
    
<br />

###  🦾 개선 포인트
- 문제 조건상 1000 이상은 검사 하지 않기에, 3자리까지만 검사.
  - 문제 잘 읽기
- 직관적인 변수명으로 개선
    
<br />

###  💡 배운 점
조건이 단순해 보여도 규칙을 정확히 정의하고, 작은 입력 범위에서는 완전 탐색이 효율적일 수 있음.
