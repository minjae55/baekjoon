### ℹ️ 문제 정보
  - 번호 : 1252
  - 난이도 : Bronze
  - 분류 : 수학, 문자열
    
<br />

### 🔎 문제 접근
입력으로 주어지는 두 수는 이진수 문자열이다. 정수로 변환 후 더한 뒤, 다시 이진수 문자열로 출력하면 됨
처음에는 Integer로 사용했지만 메모리 초과로, BigInteger를 사용하게 됨.

<br />

### 📝 사용한 자료구조
- String
  - 이진수를 문자열 형태로 입력 받기 위해 사용
- BigInteger
  - 큰 정수를 처리하기 위해 사용
  - BigInteger(String, 2) 생성자로 2진수 -> 10진수 변환
  - add() 덧셈 수행
  - toString92)로 10진수 -> 2진수 변환
    
<br />

###  🦾 개선 포인트
- 변수명 개선
- BigInteger 사용(처음에는 Integer사용 했었음)
    
<br />

###  💡 배운 점
BigInteger를 활용하면 매우 큰 수의 연산도 안전하게 처리할 수 있으며, 생성자를 통해 원하는 진법으로 바로 변환이 가능한 것을 깨달음.
