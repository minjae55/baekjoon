  - 번호 : 16198
  - 난이도 : Silver
  - 분류 : 백트래킹, 브루트포스
    
<br />

### 🔎 문제 접근
처음에는 구슬의 최대값을 기준으로 합계를 구하려고 하였지만, 
상황에 따라 최대값이 아닐 수 있음을 깨닫고 전체 순환을 해야하는 방식으로 수정

<br />

### 📝 사용한 자료구조
- ArrayList
  - remove(i)로 구슬 제거, add(i, value)로 구슬 복원하여 백트래킹 구현

<br />

### 🦾 개선 포인트
- 전역 변수 result 제거 후 재귀 반환값으로 처리
- 변수명, 함수명 변경

<br />

### 💡 배운 점
전역 변수보다 재귀 반환값으로 결과를 처리하는 것이 훨씬 안전하고 깔끔한 것을 알았으며,
최대값을 구한다고해서 매번 최대값을 기준으로 하는 것이 아닌, 입력값이 많지 않다면 전체 순환이 효율적일 수 있음.
또한 처음에는 누적 합계를 static으로 이용하엿지만 재귀 함수는, 여러개가 동시에 실행되기 때문에
이는 부적합하다는 것을 깨달음. 전체적으로 재귀 함수의 이해도가 높아짐.
